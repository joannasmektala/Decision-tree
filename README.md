Implementacja algorytmu drzewa decyzyjnego z nieczystością Giniego jako miarą heterogeniczności węzłów. Pierwsza wersja radziła sobie tylko ze zmiennymi binarnymi. Obecna wersja radzi sobie ze wszystkimi zmiennymi mierzalnymi i niemierzalnymi, przy czym zmienne niemierzalne należy najpierw zakodować. Następnym krokiem jest uwzględnienie podanej przez użytkownika minimalnej liczby obserwacji w liściu. Ostatnim krokiem jest uwzględnienie zdefiniowanej maksymalnej głębokości drzewa.
